# cpp-simple-database

## Toolchain
- Language: Modern C++
- Build System: xmake
- Unit Test: gtest

## Get Started
- Build: 
xmake
- Run target: 
xmake run target
- Run test: 
xmake run test

### Setup pre-push hook
- Install a few binary for C++ language: `clang-format`, `cppcheck`, `cpplint`
- Install pre-push hook:
```
pre-commit install -t pre-push
```
